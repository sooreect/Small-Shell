Program: CS344 Program 3 - Small Shell
Description: A mini shell with three built-in commands.

Instructions for running the smallsh shell:
1.	Save the smallsh.c program into a directory of choice
2.	Compile the program with GCC compiler by typing "gcc -o smallsh smallsh.c" onto the command line
3.	Run the executable by typing "smallsh" onto the command line
